Інструкція з запуску YouTube Video Downloader

Встановлення необхідних компонентів:

Встановіть Python (якщо ще не встановлено)
Встановіть необхідні бібліотеки за допомогою pip:
Copy pip install flask yt-dlp

Встановіть ffmpeg (потрібно для обробки відео)


Підготовка файлів:

Збережіть файл app.py у кореневій директорії проекту
Створіть папку templates
Збережіть файл index.html у папці templates
Створіть папку static/downloads у кореневій директорії проекту


Запуск сервера:

Відкрийте термінал/командний рядок
Перейдіть до директорії проекту
Виконайте команду:
Copy python app.py

Сервер запуститься за адресою http://127.0.0.1:5000/


Використання:

Відкрийте у браузері http://127.0.0.1:5000/
Вставте URL-адресу відео з YouTube
Виберіть тип завантаження (відео+аудіо, тільки відео, тільки аудіо)
Виберіть якість/формат
Підтвердіть завантаження
Після завершення, завантажте файл через посилання
